# Tech Context

## Texnologiyalar
*   **Runtime:** Node.js (Electron ichida).
*   **Desktop Framework:** Electron 33.
*   **Frontend Framework:** React 18.
*   **Build Tool:** Vite 6.
*   **Styling:** Tailwind CSS 3.
*   **Server Framework:** Express 5 (Lokal server uchun).
*   **Database:** Better-SQLite3.
*   **Printer:** `node-thermal-printer` (ESC/POS protokoli uchun).
*   **Websocket:** Socket.io (Real-time buyurtma yangilanishlari uchun).

## Rivojlanish Muhiti (Dev Environment)
*   **OS:** Windows.
*   **Package Manager:** npm.
*   **Linting:** ESLint.
*   **Testing:** Jest.

## Muhim Kutubxonalar
*   `electron-builder`: Ilovani distributiv (.exe) holatiga keltirish uchun.
*   `concurrently`: Dev rejimida bir vaqtning o'zida Frontend va Electronni ishga tushirish uchun.
*   `lucide-react`: Ikonkalar to'plami.
*   `recharts`: Grafiklar va hisobotlar chizish uchun.
*   `ip`: Lokal IP manzilni aniqlash uchun (Waiter App uchun kerak).
